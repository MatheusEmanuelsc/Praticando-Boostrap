Containers no Bootstrap

Os containers são elementos fundamentais no Bootstrap para estruturar e organizar o layout de forma responsiva.

1. Container Fixo

O .container possui uma largura fixa que se adapta ao tamanho da tela. Ele não ocupa 100% da largura e possui margens laterais automáticas.

Container

Este é um container de tamanho fixo.

2. Container Fluid

O .container-fluid ocupa 100% da largura do elemento pai, independentemente do tamanho da tela.

Container Fluid

Este container ocupa toda a largura disponível.

3. Container Responsivo

O .container-{breakpoint} muda de comportamento conforme o tamanho da tela, tornando-se fluido em telas menores.

Container Responsivo

Este container adapta seu tamanho conforme o breakpoint definido.

Personalização

  • Bordas: Adicione border para exibir uma borda ao redor do container.
  • Cores de Fundo: Utilize bg-cor (exemplo: bg-secondary) para alterar o fundo.
  • Espaçamento Interno: Use p- para padding (exemplo: p-3) e m- para margin (exemplo: m-3).
  • Cor do Texto: Utilize text-cor (exemplo: text-white) para alterar a cor do texto.

Conclusão

Os containers do Bootstrap ajudam a estruturar o layout de forma responsiva e organizada. Saber quando usar cada tipo de container melhora a experiência do usuário.
