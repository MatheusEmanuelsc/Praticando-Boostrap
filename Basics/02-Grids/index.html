Grid no Bootstrap

O sistema de grid do Bootstrap utiliza uma estrutura baseada em linhas e colunas para criar layouts flexíveis e responsivos.

1. Estrutura Básica

Para utilizar o grid, é necessário definir uma row (linha) como elemento pai e dentro dela adicionar os elementos filhos com a classe col (coluna).

2. Definição de Tamanhos

O grid do Bootstrap é baseado em um sistema de 12 colunas. Podemos definir o tamanho de cada coluna utilizando col-{n}, onde n é um número de 1 a 12.

3. Responsividade

Podemos definir diferentes larguras de colunas para diferentes tamanhos de tela usando breakpoints:

  • col-sm-{n} - Extra pequeno (≥576px)
  • col-md-{n} - Pequeno (≥768px)
  • col-lg-{n} - Médio (≥992px)
  • col-xl-{n} - Grande (≥1200px)
  • col-xxl-{n} - Extra grande (≥1400px)

4. Alinhamento

Podemos alinhar os elementos dentro de uma linha usando classes utilitárias:

  • justify-content-start - Alinha à esquerda
  • justify-content-center - Centraliza
  • justify-content-end - Alinha à direita
  • align-items-start - Alinha no topo
  • align-items-center - Alinha no centro verticalmente
  • align-items-end - Alinha na base

Conclusão

O sistema de grid do Bootstrap facilita a criação de layouts responsivos e flexíveis, permitindo organizar elementos de forma eficiente em diferentes tamanhos de tela.
