Colunas no Bootstrap

O sistema de colunas do Bootstrap permite organizar e distribuir os elementos de forma flexível e responsiva.

1. Colunas Iguais

Quando usamos apenas col, as colunas têm o mesmo tamanho e se ajustam automaticamente.

2. Colunas com Largura Definida

Podemos definir a largura das colunas usando col-{n}, onde n vai de 1 a 12.

Personalização

  • Espaçamento: Use m- para margem e p- para padding.
  • Cores: Utilize bg-cor (exemplo: bg-primary) para cores de fundo.
  • Quebras: O w-100 força uma quebra de linha.

Conclusão

O sistema de colunas do Bootstrap facilita a criação de layouts responsivos e flexíveis, adaptando-se a diferentes tamanhos de tela.
