Margens, Offsets e Gutters no Bootstrap

Esses recursos permitem ajustar o espaçamento entre elementos e o alinhamento dentro do sistema de grid.

1. Margens

As margens podem ser aplicadas utilizando as classes m-* (margem externa) e p-* (margem interna - padding).

2. Offset

O offset é utilizado para deslocar colunas dentro do grid.
