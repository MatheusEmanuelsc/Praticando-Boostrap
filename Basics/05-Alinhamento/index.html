Alinhamento no Bootstrap

O Bootstrap fornece diversas classes para alinhamento de elementos dentro do layout, utilizando flexbox e alinhamento de texto.

1. Alinhamento Vertical

Para centralizar elementos verticalmente dentro de um contêiner flexível, utilize align-items-*.

Elemento centralizado verticalmente

2. Alinhamento Horizontal

Para alinhar elementos horizontalmente dentro de um contêiner flexível, utilize justify-content-*.

Elemento centralizado horizontalmente

3. Alinhamento de Texto

O alinhamento do texto pode ser definido com text-start, text-center e text-end.

4. Alinhamento em Grid

No sistema de grid do Bootstrap, podemos alinhar colunas usando align-self-* ou align-items-* na linha.

Conclusão

O Bootstrap torna o alinhamento de elementos mais fácil e flexível, permitindo organizar layouts responsivos de forma prática.
