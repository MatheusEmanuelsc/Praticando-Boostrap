Imagens no Bootstrap

1. Imagem Responsiva

Utilize a classe img-fluid para tornar a imagem responsiva.

2. Imagem Arredondada

A classe rounded aplica bordas arredondadas à imagem.

3. Imagem em Miniatura

Use img-thumbnail para adicionar uma moldura arredondada à imagem.

4. Tamanho Personalizado

Utilize classes de largura (w-50) e altura (h-50) para definir tamanhos personalizados.
