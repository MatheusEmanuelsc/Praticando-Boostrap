2. Tabela com Estilos

Podemos adicionar diferentes estilos à tabela usando classes do Bootstrap:

  • table-dark: Aplica um tema escuro.
  • table-striped: Adiciona linhas com cores alternadas.
  • table-hover: Destaca a linha ao passar o mouse.
  • table-bordered: Adiciona bordas a todas as células.
  • table-borderless: Remove todas as bordas.
  • table-sm: Reduz o espaçamento entre as células.
