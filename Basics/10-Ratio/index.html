Classe `ratio` no Bootstrap

O Bootstrap oferece a classe `ratio` para criar elementos com proporções fixas e responsivas, como vídeos, imagens e outros elementos.

1. Utilizando a Classe `ratio` para Vídeos

Para tornar vídeos responsivos, você pode usar a classe `ratio` junto com a classe de proporção desejada, como `ratio-16x9` ou `ratio-4x3`.

2. Utilizando a Classe `ratio` para Imagens

A classe `ratio` também pode ser usada para manter proporções fixas de imagens.

Conclusão

A classe `ratio` do Bootstrap é uma maneira simples e eficaz de garantir que seus vídeos e outros conteúdos embutidos se ajustem ao tamanho da tela de maneira responsiva, mantendo as proporções fixas e evitando distorções.
