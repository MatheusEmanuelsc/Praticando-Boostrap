📏 Padding e Margin no Bootstrap

🔹 Margin (`m-`)

As classes `m-` são usadas para adicionar margens externas ao elemento.

Margin 4 (16px)
Margin 0 (sem margem)
Centralizado horizontalmente

🔹 Padding (`p-`)

As classes `p-` são usadas para definir preenchimento interno do elemento.

Padding 5 (grande espaçamento interno)
Padding 0 (sem espaçamento interno)
Padding horizontal 3 e vertical 5

🔹 Combinação de Padding e Margin

É possível combinar `m-` e `p-` para um espaçamento mais preciso.

Margin 3 + Padding 4
