Classes de Botões do Bootstrap:

  • btn btn-*: Define a cor do botão (como btn-primary, btn-secondary, etc.).
  • btn-outline-*: Para criar botões com borda e cor de fundo transparente.
  • btn-lg: Para botões maiores.
  • btn-sm: Para botões menores.
  • btn-block: Para botões que ocupam toda a largura do contêiner.
  • disabled: Para desabilitar um botão, tornando-o não clicável.
