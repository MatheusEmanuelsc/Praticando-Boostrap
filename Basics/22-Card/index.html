Para utilizar a paginação é necessário utilizar a classe pagination no contêiner.

Cada item dentro da lista de páginas deve ter a classe page-item, e os links dentro dos itens devem ter a classe page-link.

Você também pode adicionar a classe active a um item de página para indicar a página atual, ou a classe disabled para desabilitar um link (por exemplo, o link "Anterior" na primeira página ou "Próxima" na última).
