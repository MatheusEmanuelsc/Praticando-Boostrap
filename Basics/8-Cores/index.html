Cores no Bootstrap

O Bootstrap oferece classes para estilização de cores tanto em textos quanto em fundos de elementos.

1. Cores de Texto

Para alterar a cor do texto, utilize as classes text-*. Exemplo:

Muted (cinza claro)

Primary (azul)

Success (verde)

Info (azul claro)

Warning (amarelo)

Danger (vermelho)

Secondary (cinza escuro)

White (branco)

Dark (preto/cinza escuro)

Light (cinza claro)

2. Cores de Fundo

Para definir cores de fundo, use as classes bg-*. Exemplo:

Fundo azul (Primary)
Fundo verde (Success)
Fundo amarelo (Warning)
Fundo vermelho (Danger)
Fundo azul claro (Info)
Fundo cinza escuro (Secondary)
Fundo preto (Dark)
Fundo cinza claro (Light)

3. Opacidade

Podemos adicionar opacidade usando -50 nas cores de fundo. Exemplo:

Primary com 50% de opacidade
Danger com 50% de opacidade

Conclusão

O Bootstrap facilita a personalização de cores para texto e fundo, permitindo ajustes rápidos no design da aplicação.
